# Product Explorer — recruitment task app

This repository contains a ready React + Vite + TypeScript app prepared for a QA Automation Engineer recruitment task.

## Goal

The task is designed for a middle / mid+ QA Automation candidate who should demonstrate:

- React and TypeScript understanding,
- Cypress E2E testing,
- Cypress Component Testing,
- API testing and network assertions,
- code structure and maintainability.

## Tech stack

- React
- Vite
- TypeScript
- Cypress
- DummyJSON as a free public API

## Local run

```bash
npm install
npm run dev
```

In a second terminal:

```bash
npm run cy:open
```

or

```bash
npm run test:e2e
npm run test:component
```

## Notes

- The frontend app is complete and ready for testing.
- The candidate is expected to focus on Cypress tests, not on implementing new frontend features.
- Detailed requirements are described in `TASK.md`.
